


# AI工具导航

## 项目介绍
这是一个专业的AI工具资源导航平台，旨在为AI工具使用者、开发者和研究人员提供便捷的工具查找和使用体验。

## 功能特点

- 简洁现代的界面设计
- 响应式布局，支持多端访问
- 实时搜索过滤功能
- 分类导航系统
- 工具卡片展示
- 标签筛选功能
- 收藏功能
- 访问统计
- 平滑过渡动画

## 技术栈

- HTML5
- CSS3 (Grid/Flexbox布局)
- JavaScript (原生)
- LocalStorage (数据持久化)

## 目录结构

```
├── index.html      # 主页面
├── README.md       # 项目说明文档
└── assets/         # 资源文件夹（如有需要）
```

## 部署说明

1. 克隆仓库：
   ```bash
   git clone <repository-url>
   ```

2. 直接打开index.html文件即可在浏览器中查看网站

## 开发指南

### 添加新工具

在`index.html`文件中的`toolsData`数组中添加新的工具对象，格式如下：

```javascript
{
    name: "工具名称",
    description: "工具描述",
    tags: ["标签1", "标签2"],
    category: "所属分类",
    heat: 评分（0-10）
}
```

### 自定义样式

样式变量定义在`:root`选择器中，可以通过修改这些变量来自定义主题：

```css
:root {
    --primary-color: #4a90e2;
    --secondary-color: #50e3c2;
    --text-color: #333;
    --light-bg: #f8f9fa;
}
```

## 贡献指南

### 提交新工具

如果您想要向导航网站提交新的AI工具，请确保提供以下信息：

#### 必填字段

- **工具名称**：简洁明了的名称
- **工具描述**：100字以内的功能介绍
- **工具链接**：有效的访问地址
- **所属分类**：选择最合适的分类（AI绘画/AI写作/AI编程等）
- **标签**：2-5个关键词标签
- **访问类型**：免费/付费/会员制

#### 提交格式示例

```json
{
    "name": "ChatGPT",
    "description": "OpenAI开发的大型语言模型，能够进行自然对话、文本创作、代码编写等多种任务",
    "url": "https://chat.openai.com",
    "category": "AI对话",
    "tags": ["语言模型", "对话", "写作"],
    "accessType": "付费"
}
```

您可以通过以下方式提交：
1. 在GitHub上提交Pull Request
2. 在Issues中提供工具信息
3. 发送邮件至管理员

我们会在收到提交后的3个工作日内进行审核和处理。
